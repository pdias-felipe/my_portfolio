Sobre Mim

Um constante aprendiz apaixonado pelo conhecimento, passando por uma transição de carreira, da Filosofia para a área de Tecnologia, para ser mais específico, Desenvolvimento Web. Durante 7 anos na área da Educação como professor, tive a oportunidade de desenvolver diversas habilidades e competências que agora, quero trazer para essa nova área de atuação. Por exemplo, trabalhar em equipe (tendo em vista que o trabalho pedagógico só é bem-sucedido se feito em conjunto), gestão do tempo (uma aula tem tempo pré-determinado e esse deve ser aproveitado da melhor maneira possível), cumprimento de prazos (relatórios, planilhas, resultados são sempre exigidos nessa profissão), raciocínio lógico (afinal, lecionei por anos Filosofia e Sociologia, competência básica para tal exercício). Sem contar a enorme vontade de sempre aprender coisas novas. O universo da tecnologia é incrível e tenho me aprofundado bastante em aprender novas linguagens, frameworks e tudo mais que vier por aí.

CONHECIMENTOS

HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto") é uma linguagem de marcação utilizada na construção de páginas na Web.

CSS

CSS é a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código criado por linguagens como HTML, XML ou XHTML, por exemplo. De forma prática, ela funciona como uma camada de personalização ao conteúdo visível.

JAVASCRIPT

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web.

BOOTSTRAP

Bootstrap é um framework front-end que fornece estruturas de CSS para a criação de sites e aplicações responsivas de forma rápida e simples. Além disso, pode lidar com sites de desktop e páginas de dispositivos móveis da mesma forma.

REACT.js

O React é uma biblioteca JavaScript declarativa, eficiente e flexível para criar interfaces com o usuário. Ele permite compor UIs complexas a partir de pequenos e isolados códigos chamados “componentes”.
